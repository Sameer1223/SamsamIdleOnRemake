# Sameer IdleOn Remake

Build Located At: https://sameerk9.itch.io/samsam-idleon-clone

Demo Video: https://youtu.be/OMmwnIEz-c8

## Features
- Combat
- Auto Combat
- Offline Rewards
- Player Health / Mana / XP
- Mana Ability Power Strike -> 2x Damage on the Next Attack, Used in auto comboat as well
- Enemy Mobs with Combat Damage, Health, Drops
- Upgrades that can be purchased with coins
- Talent upgrades that can be purchased with talent points (1 per level)
- Interactable inventory and hotbar
- Crafting menu and recipes to create more complex items with stat bonuses
- Hotbar to equip crafted items
- Animations and Sprites courtesy of GandalfHardcore Free Asset pack
- Mining skill to generate ores
- Final boss quest with a damage mechanic

## Future Add Ons
