# Samsam IdleOn Remake

This folder contains all assignment-specific assets, code, prefabs, scenes, and documentation.

## Structure

- `Art/` - Imported free 2D assets, generated placeholder art, and sprites.
- `Data/` - ScriptableObject assets for characters, enemies, items, skills, quests, recipes, maps, and progression.
- `Docs/` - Design notes, architecture notes, asset credits, and submission support material.
- `Prefabs/` - Gameplay and UI prefabs.
- `Scenes/` - Game scenes owned by this assignment.
- `Scripts/Runtime/` - Runtime gameplay code.
- `Scripts/Tests/` - Edit mode and play mode tests.
- `Settings/` - Assignment-specific Unity settings assets.
- `UI/` - UI assets and reusable UI prefabs.
- `VFX/` - Visual effects assets.
